## Parte 2 — Análise de Mercado 2026

### Competidores Diretos

#### LangChain / LangGraph / CrewAI / AutoGen
- **Modelo:** Open-core + cloud pago
- **Preços:** LangSmith ~$39/mo, CrewAI cloud ~$40/mo
- **Problemas:**
  - Custo inflado
  - Abstração tóxica
  - Produção quebra
  - Equipes abandonam
  - 40% dos projetos morrem abandonados (dados reais de 50 projetos analisados)
- **Nosso diferencial:** Governança RuleOps + pre-commit rígido + zero lock-in

#### Redis / MongoDB / HashiCorp
- **Histórico:** Abandonaram modelo 100% open-source
- **Motivo:** Não paga contas
- **Mudança:** Source-available (backlash enorme)
- **Lição:** Monetização de código livre puro é ilusão

#### Red Hat / LangSmith
- **Modelo:** Serviços (suporte, hosted, enterprise)
- **Sucesso:** Pagam contas
- **Lição:** Venda serviços, não código

### Comparação de Diferenciais

| Dimensão | LangChain/CrewAI | EGOS Commons |
|----------|-----------------|--------------|
| Preço | $39-40/mo (cloud) | Grátis (código) + serviços pagos |
| Liberdade/Customização | Média | Alta (self-hosted) |
| Tempo de implantação | Médio | Baixo (implementação pronta) |
| Dependência de fornecedor | Alta | Zero |
| Transparência tecnológica | Média | Alta |
| Governança rígida | ❌ | ✅ RuleOps |
| Risco de lock-in | Alto | Zero |

## Parte 3 — Riscos Jurídicos e Operacionais

### Risco 1: Cashback 95% + Compra ETC (CRÍTICO)

#### Problema
- **Brasil 2026:** Banco Central estendeu regras AML para crypto em fevereiro
- **Split payments:** Combate fraude de VAT
- **Qualquer "pague 100, devolvo 95 automático" + crypto = bandeira vermelha automática**

#### Consequências Reais
- PagSeguro/Stripe bloqueiam
- Receita Federal audita
- Contas congeladas
- Multas

#### Casos Reais 2025
- Empresas que tentaram cashback crypto foram multadas
- Algumas tiveram contas congeladas

#### Psicologia do Comprador
- "Por que pagar 100 pra ganhar 95? É truque de preço inflado"
- Ninguém confia

#### Decisão
❌ **ELIMINAR do MVP**  
✅ Testar depois de validar com advogado

### Risco 2: Ironia Anti-Capitalista no Branding

#### Problema
- Visitante médio pensa: "Isso é troll? Golpe? Movimento político?"
- Conversão cai 80-90%

#### Dados Reais
- Projetos que tentaram branding anti-capitalista em OSS viraram meme ou morreram em 3 meses

#### Decisão
❌ **ELIMINAR do branding principal**  
✅ Guardar para blog/manifesto

### Risco 3: Escambo/Reciprocidade Complexo

#### Problema
- Valorar "feedback", "melhoria", "tutorial gravado"?
- Quem arbitra?
- Como evitar brigas?
- Como escalar sem moderação?

#### Casos Reais
- Gaia-X (maior projeto europeu de data commons) virou papelaria
- Barter real em tech OSS: zero casos de sucesso em escala
- Comunidades que tentaram viraram ghost towns

#### Decisão
❌ **ELIMINAR escambo complexo do MVP**  
✅ Usar pontos simples (GitHub stars, issues, PRs)

### Risco 4: Orquestração Automática Hype

#### Problema
- Atrian/ClawFlows ainda são esqueleto
- Não existe orquestração consolidada em produção
- Prometer "agentes entregam tudo" = over-promise suicida

#### Consequência
- Cliente compra, espera mágica
- Recebe bug + suporte manual
- Conversão vira reclamação + chargeback

#### Decisão
❌ **ELIMINAR hype de orquestração automática**  
✅ Ser honesto: "por enquanto entregamos via humanos + agents assistidos"
