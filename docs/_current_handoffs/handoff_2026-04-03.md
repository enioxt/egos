## 📚 KNOWLEDGE DISSEMINATED

### Learnings to Save to HARVEST.md:
1. **Error Handling Pattern**: Subprocess error handling in Bun
   - Capture stderr via `proc.stderr` → `new Response(proc.stderr).text()`
   - Always check exit code, not just file existence
   - Write error reports to disk for debugging (not just stdout)

2. **Path Resolution**: Relative paths in Bun from import.meta.dir
   - `import.meta.dir` = directory of current file
   - Calculate depth carefully: each `..` = one parent level
   - Test with `path.normalize()` before deploying

3. **LLM Provider Fallback Chain**:
   - Multiple tiers: fast (qwen-flash), default (qwen-plus), deep (qwen-max)
   - Catches AUTH_ERROR immediately (don't continue chain)
   - Retries on RATE_LIMIT to next model
   - All models in one file: llm-provider.ts
