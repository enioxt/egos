# Cross-Reference Visualizer

**Status**: ACTIVE

**Path**: `scripts/cross_reference/cross_reference_visualizer.py`

**Category**: Visualization

**Maintainer**: EGOS Development Team

## Description

Visualizes cross-references across the EGOS ecosystem by creating interactive network graphs and Mermaid diagrams. Works with the output from the cross-reference validator to provide visual insights into document relationships.

### Example 1: Generate visualizations from latest validation report

```bash
python scripts/cross_reference/cross_reference_visualizer.py
```
