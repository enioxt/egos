## 💡 Decision Framework

**Quando receber uma tarefa, faça essas perguntas:**

```
1. É RÁPIDO? (< 100ms necessário?)
   YES → Use Gemini Flash
   NO  → Continue

2. É CÓDIGO?
   YES → Use Qwen3 Coder OU LLaMA
   NO  → Continue

3. Precisa REASONING profundo?
   YES → Use LLaMA 3.3 70B
   NO  → Continue

4. Precisa conhecimento 2026?
   YES → Use EXA + Gemini 5.4 Pro
   NO  → Continue

5. É CRÍTICO (sem compromissos)?
   YES → Use Claude Opus
   NO  → Continue

6. Tem muita CONTEXT (100K+ tokens)?
   YES → Use Gemini 1.5 Flash (1M context)
   NO  → Continue

7. Precisa MULTIMODAL (imagem)?
   YES → Use Gemini Flash
   NO  → Continue

DEFAULT → Use LLaMA 3.3 70B (melhor ratio)
```
