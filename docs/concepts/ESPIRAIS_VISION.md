## Core Concept

**"Listening Spirals"** — Collaborative environments where humans converse
with multiple specialized AIs that ask Socratic questions, bring diverse
perspectives, and help you **think better**, not just execute.

## Differentiator

If Figma is "Google Docs of design", **EGOS is the Figma of decisions**.

| Aspect | ChatGPT | EGOS Espirais |
|--------|---------|---------------|
| Quality | Generic 1-step response | Multiple specialized perspectives |
| Process | None | Embedded (Discovery → Strategy → Action) |
| Method | Answers | Socratic questions (maieutic) |

## 3-Column Layout

1. **Navigation** — Choose specialist agents
2. **Conversation** — Collaborative chat
3. **Context** — Real-time insights, files, status

## 8 Specialist Agents (planned)

Analyst, Ethical (ATRiAN), Creative, Technical,
Mediator, Researcher, Educator, Critic

## Engineering Standards (from v5)

- API-First: Every feature is an API first, UI just calls it
- Agent Relay: Agents can spawn other agents (Swarm Intelligence)
- E2E testing priority (Playwright), not just unit tests
- Mobile-first design (absolute requirement)
